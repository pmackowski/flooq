This document is the API specification for the Flooq library.

Flooq is a reactive API for RabbitMQ based on Project Reactor.

Flooq

        Flooq flooq = new FlooqBuilder()
                .fromTopic(consumer -> consumer
                        .inputExchange("my.exchange")
                        .queue("my.queue")
                        .routingKey("#")
                        .consumeNoAck(messages -> messages
                                .map(this::toUppercase)
                        )
                )
                .build();

        Disposable disposable = flooq.start();

More detailed documentation is available in the Reference Guide.
